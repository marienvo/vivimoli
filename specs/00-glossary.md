# Glossary

- **world**: de engine state container voor de simulatie.
- **entity**: id binnen ECS.
- **component**: data bucket gekoppeld aan entities.
- **system**: pure update-stap over ECS data in vaste volgorde.
- **tick**: een fixed-step simulatiestap.
- **snapshot**: read-model voor render/UI.
- **command**: typed write request naar engine.
